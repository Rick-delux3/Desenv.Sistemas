Bem-vindo ao Sistema de Cadastro de Países da ONU

Sobre:

Este é um projeto desenvolvido para simular um sistema de gerenciamento de informações sobre os países membros da Organização das Nações Unidas (ONU).

  • Cadastrar novos países utilizando dados reais da API RestCountries;
  • Visualizar todos os países cadastrados em Cartões dinamicos
  • Editar as informações de um país utilizando seu código de registro;
  • Excluir registros de forma prática e segura.

Sobre a ONU

A Organização das Nações Unidas é uma entidade internacional criada em 1945, com o objetivo de promover a paz, a segurança, os direitos humanos e o desenvolvimento sustentável em todo o mundo. Composta por quase todos os países do planeta, a ONU é um símbolo de cooperação internacional.

Explore!!

Use o menu de navegação para acessar a área de cadastro de países, visualizar a lista de membros registrados e interagir com todas as funcionalidades do sistema.
